# Prime Day

年(西暦)、月、日を`yyyy-mm-dd`の8桁で表現したもののうち素数であるものを`prime_day`と呼ぶ。
与えられた2つの日付の間に存在する`prime_day`の数を出力せよ。

## Input
入力は以下の形式で与えられる
```
A B
```
ただし、$A, B$は以下の条件を満たす
- $19000101 \le A \le B \le 21001231$

## Output
$A$以上$B$以下の`prime_day`の総数を出力せよ。
ただし、うるう年は存在せず**全ての年の全ての月**において1日から31日までが存在するものとする。
