# FizzBuzz

## 問題
$1$ 以上の整数 $N$ が与えられます。

$1$ 以上 $N$ 以下の整数 $i$ について順に、以下の文字列を出力してください。

1. $i$ が $3$ で割り切れるとき `Fizz`
1. $i$ が $5$ で割り切れるとき `Buzz`
1. $i$ が $3$ と $5$ の両方で割り切れるとき、 `FizzBuzz`
1. それ以外のとき、$i$ 自身

## 制約
- $1 \le N \le 100$

---

## 入力
入力は以下の形式で標準入力から与えられる。

```md
$N$
```

## 入力例1
```
5
```

## 出力例1
```
1
2
Fizz
4
Buzz
```

---

## 入力例2
```
16
```

## 出力例2
```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
```
